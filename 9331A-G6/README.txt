This folder contains 2 text files and 1 folder
	README.txt
	LOGS.txt - contains the commit history in the repository
	website - contains all the files for the website

HOW TO ACCESS THE WEBSITE
	1. Open the folder named website.
	2. Look for html file named index.
	3. Open it using a browser.
	4. You will see the Course Title and a button on its right.
	5. When you click the button, you will be directed to the page about the World Wide Web
	6. You can use the menu to access different topics of the course.
	7. There is also about menu that contains information about the writers of the website.
