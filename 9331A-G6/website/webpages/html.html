HyperText Markup Language

  • Application of Standard Generalized Markup Language or SGML (ISO 8879)
  • Structure & content of web pages
  • Today, HTML is for semantic specifications of the webpage structure and content with presentational aspects being delegated to stylesheets.
    • Initially developed at CERN, then IETF, then W3C and WHATWG

Version History

Pre standardization:

  • HTML tags by Tim Berners-Lee, late 1991
  • HTML tags, Internet Draft, by Tim Berners-Lee and Dan Connolly at IETF, mid 1993
  • HTML tags,(hypertext markup format) Internet Draft by Dave Raggett, late 1993
