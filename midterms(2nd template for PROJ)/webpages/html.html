HyperText Markup Language

Hypertext Markup Language is an applicaton of Standard Generalize Markup Language (SGML). It is used to form the structure of web pages and its contents. Initially developed at CERN, then IETF, then W3C and WHATWG. Today, HTML is for semantic specifications of the webpage structure and content with presentational aspects being delegated to stylesheets.

  • HTML tags, Internet Draft, by Tim Berners-Lee and Dan Connolly at IETF, mid 1993
  • HTML tags,(hypertext markup format) Internet Draft by Dave Raggett, late 1993

Document Structure

  • Version information - contains the information of html version that you have used. (i.e. <!DOCTYPE html>)
  • Root - it contains different tags that will be in your webpage. (<html> </html>)
    • "<head> </head>" - it is where you can put the title, metadata, some links (stylesheet, icon, etc.)
    • "<body> </body>" - it defines the body of the document. It is where the contents of your html is places. ( text, images, links, and etc.)
  • Document Type Declarations
  • Elements
    • Tags
    • Content
    • Attributes and Attribute Values
      • Global Attributes
      • Custom Non-Visible data attribute
      • WAI-ARIA attributes
      • Element specific attribute
  • MATHML
    • Mathematical Markup Language
    • Allows you to add mathematical symbols/ notations properly
  • Character References
    • These are character representations that is preceded with & and ended with ;
  • Comments
    • <!-- -->
    • Some comments have different function just like in java where they have javadoc
