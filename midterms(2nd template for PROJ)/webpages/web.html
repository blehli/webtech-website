WORLD WIDE WEB

What is World Wide Web (WWW)?

- World Wide Web is a technology invented by SIR TIM BERNERS-LEE during the year of 1989

- It is a collection of WEB RESOURCES and APPLICATIONS

- The applications allows the client access the internet through a WEB BROWSER

- The server HOSTS WEB RESOURCES for the clients to access it


How to access the World Wide Web?

- The client can access the World Wide Web by using a web browser.
After opening a browser, the client will type a URL address for him/her to access the resource

Web Browser - It is a software application that the client uses to retrieve resources
from a specific URL

Uniform Resource Locator (URL) - It is a unique identifier for a specific resource.

Parts of a URL

http://webtek.org/index.html

- Above is an example of a URL link which is divided into parts.

http:// -It is an acronym for HyperText Transfer Protocol. It is a protocol used to connect the internet

webtek - It is the DNS or the Domain Name Server. It is the readable and easy to memorize addresses that will be translated into Internet Protocols

index.html - This is the resource that you want to retrieve
